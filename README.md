# Test-Data Generator

This project is a generator for test data where the results need to have a specific format or pass a checksum test (e.g., IBAN).
Everything is generated directly in the browser (even the download files), you don't need any backend to run this app.
I case you want to try out the test-data generator, see the links below.

# Quick Start

The usual commands for Node projects can be used in this project.

- `npm ci` to install all dependencies
- `npm start` to run the app locally for development
- `npm run clean` to delete all generated files (build output etc.)
- `npm run format` to format the sources
- `npm run lint` to run the linter
- `npm run build` to build the app for production
- `npm test` to run unit tests
- `npm run e2e` to run E2E tests
